Convert biblatex files to bibtex.

**Note: you need [`biber`](https://github.com/plk/biber).**

Usage:

`biblatex2bibtex /path/to/biblatex_file1.bib /path/to/biblatex_file2.bib --output /path/to/bibtex_file.bib`
