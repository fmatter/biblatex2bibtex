Convert biblatex files to bibtex.


Usage:

`biblatex2bibtex /path/to/biblatex_file.bib` (will create `/path/to/biblatex_file_out.bib`)

or 

`biblatex2bibtex /path/to/biblatex_file.bib /path/to/bibtex_file.bib`
